FamilyCompass

For privacy-conscious families & military families

The FamilyCompass app combines real-time GPS location sharing, encrypted messaging, a shared calendar, and a one-tap SOS alert button — all end-to-end encrypted. Built on a zero-knowledge architecture: your family's location is never accessible to us or anyone else.

  • Real-time GPS & compass location sharing for every family member
  • End-to-end encrypted messaging
  • Shared family calendar with intelligent conflict detection
  • Encrypted photo sharing
  • Smart grocery lists — notified automatically when a supermarket is detected via GPS
  • Shared family grocery lists & task management across every household member
  • Home Dashboard covering gardening, nutrition, household tasks and more
  • Personal User Dashboard with attention to physical & mental wellbeing, hobbies, and daily routines
  • Works on iOS & Android — pure React Native

One Encrypted Platform. Four Products.

All four AI Pieces products share a single encrypted codebase — delivering high code reuse and consistent security engineering across every sector we serve.

🔐

AES-256 End-to-End Encryption

Built with ECDH key exchange and the Signal Protocol, and hardware-backed key storage via iOS Secure Enclave and Android StrongBox. Designed around a zero-knowledge architecture — keys stay on your device, so AI Pieces cannot read your data.

🏗️

Shared Encrypted Codebase

React Native + Supabase backend. One platform serves four products — faster iteration, shared security hardening, and consistent quality across every vertical.

🌍

GDPR Compliant

Built in Kerry, Ireland — inside the EU. GDPR-compliant from day one across all products. HealthCompass is designed to meet HIPAA standards.

📡

Real-Time Everywhere

Supabase real-time subscriptions power live GPS, live messaging, and live calendar updates across all four products. Offline sync ensures functionality even without connectivity.

🛡️

Intrusion Detection Built In

Built with canary token monitoring, honeypot records, and Row Level Security policies designed to flag unauthorised access attempts across every product.

📱

iOS & Android Native

Pure React Native — no shortcuts. Hardware-backed biometric authentication. Tested and deployed across both major platforms with mobile-first as a non-negotiable standard.
